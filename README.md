## instalation
```sh
git clone kibana
```
```sh
cd kibana
```
```sh
yarn
```

## list npm scripts 
- ```sh yarn / npm server``` to load server http server
- ```sh yarn / npm test``` to run all test
- ```sh yarn / npm test:list``` to show all tests 
- ```sh yarn / npm tests:resolve``` generate file with stats
